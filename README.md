# Hadoop Ansible
This ansible playbook contains roles for the following technologies: 
* Apache Hadoop
* Apache Spark
* Apache Storm
* Apache Zookeeper
* Apache Kafka
* Apache Pig
* Apache Hive
* ElasticSearch

Note: This ansible module is not ready yet and is still in development. 



## Vagrant
```bash
vagrant plugin install vagrant-env
vagrant up
```
